# 版本记录

## UI/体验阶段更新 - 2026-06-11

- 新增：
  - 浏览器本地自定义头像。
  - 统一样式的轻量自定义下拉菜单增强。
- 修复：
  - 用户头像/昵称区域与标题重叠的问题。
  - 日期输入占位中中英文混用的问题。
  - 主题按钮使用 emoji 与整体视觉不一致的问题。
  - 最新上传的大图不再因入场缩放、照片滤镜或强制 GPU 合成而显得发糊。
- 调整：
  - 项目定位从偏家庭相册调整为更开放的回忆相册。
  - 首页、上传整理区、系统状态区和工具栏改为更简约克制的视觉风格。
  - 描述模板和输入占位改为更轻快的表达。
  - 上传和浏览链路改为清晰度优先：100MB 内保留原图，主画廊大图、最近上传记忆卡和时间线封面优先展示原图。
  - 缩略图升级为 1600px 高质量预览，旧低清缩略图会自动重建。
- 注意：
  - 头像只保存在当前浏览器，不上传服务器、不多设备同步。
  - 原图优先会让大图展示更清晰，但高分辨率照片很多时加载量会比缩略图方案更高。
  - 本次保留原生前端和文件 JSON 存储边界，没有引入框架或账号系统。
