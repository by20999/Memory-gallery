## 当前工作区注意

截至本次文档和加固工作，Git 工作区中已有业务代码未提交改动，包括故事流、写入鉴权、写入队列、脚本和文档等。后续不要回滚这些改动，除非你明确要求。

当前已知上线前注意事项：

- 用户已确认旧相册数据不保留，当前照片元数据已清空，`npm run audit:data` 通过。
- 部分历史界面文案仍建议再做一次 UTF-8 和用户可见文本清理；本轮已将“家庭”定位文案改为更开放的“回忆相册”。
